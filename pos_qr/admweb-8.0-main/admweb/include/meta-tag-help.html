title
เป็นข้อมูลในส่วนของ title bar หรือ คำอธิบายหัวของหน้าเว็บนั้นๆครับ ควรใช้คำหรือ ประโยคไม่ยาวมากนัก ไม่เกิน 255 ตัวอักษร ที่สื่อกับสินค้า หรือ ประเภทของเว็บครับ

description
เป็นข้อความอธิบายเว็บไซต์หรือหน้านั้นๆเกี่ยวข้องกับเรื่องใด คำอธิบายไม่ควรเกิน 255 ตัวอักษรครับ

keywords
เป็นคำหลักของเว็บไซต์ หรือ ของสินค้า บริษัทนั้นๆ เพื่อให้การค้นหาของ web search เข้าถึงได้ง่ายขึ้น เช่นทำเว็บเกี่ยวกับสัตว์เลี้ยงข้อมูลที่ใส่จะเป็นดั่งตัวอย่างด้านล่างนี้ครับ

ตัวอย่าง : สัตว์เลี้ยง,สุนัข,การเลี้ยงสุนัข,การเลี้ยงสัตว์,ขายสัตว์เลี้ยง,บทความสัตว์เลี้ยง,ความรู้เรื่องสัตว์

ดั่งตัวอย่างจะเห็นได้ว่าในคำแต่ละคำจะแยกกันด้วยเครื่องหมาย ( , ) และในการใส่ข้อมูลไม่ควรใส่คำซ้ำกัน รวมไปถึงไม่ควรใส่คำที่ไม่เกี่ยวข้องกับเว็บของเรา

robots และ googlebot
เป็นการกำหนดว่าเราต้องการให้ botgoogle มาเก็บข้อมูลบนหน้าเว็บของเราหรือไม่ กรณีที่เปิดให้เข้ามาเก็บจะใส่เป็น ( index, follow ) ซึ่งเป็นค่าพื้นฐานของระบบเราอยู่แล้วครับ กรณีที่ไม่ต้องการให้ bot เข้ามาเก็บข้อมูลให้ใส่เป็น ( noindex, nofollow ) ทางเราแนะนำให้ใส่ตามค่าพื้นฐานของระบบเราไว้ครับ หรือ จะสามารถใส่เป็น all ก็ได้เช่นกันครับ

contact_addr
สำหรับใส่อีเมล์ของทีมงาน ให้คนติดต่อกลับอาจจะใส่เป็นอีเมล์ของบริษัทของท่านหรืออีเมล์ของเจ้าของเว็บก็ได้เช่นกันครับ

copyright
ใส่พวกชื่อบริษัทแสดงความเป็นเจ้าของเว็บไซต์ หรือ บริษัท รวมไปถึงผู้จัดทำโปรแกรม ผู้เขียนโปรแกรมก็ได้เช่นกันครับ

author
ใส่ชื่อเจ้าของโปรแกรม หรือ เจ้าของเว็บไซต์ครับ

revisit-after
สำหรับสั่งให้ Search engine กลับมาเก็บข้อมูลที่เว็บไซต์ อีกครั้งในกี่วันข้างหน้าครับหน่วยจะเป็นวันครับ ระบบพื้นฐานจะใส่ไว้เป็น 1 วันครับ
